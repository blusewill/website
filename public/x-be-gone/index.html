So Twitter have their New logo and Name Called “X”

The vision for X has been unfolding over the last 9 months – and there’s so much more on the horizon. Today we’re taking a moment to celebrate the progress and thank our teams and all our partners. https://t.co/azXPbe8IpO

In this website Article I am going to show you how can you change it from “X” to Twitter

Installation

Here is the installation of the extension

Firefox

Firefox User is probably the most easiest one which you have to do is go to Firefox Add-on and install this Extension after the Installation go to twitter and you should see it switch to twitter instead of X

Chromium Based Browser

So if you are using Chrome/Brave that is based on Chromium you have to install manually through the Source

Click on < > Code and Download zip and Extracted into a Folder

Now go to Extension Tab by typing chrome://extensions in the URL bar

And you will see Developer Mode on the Top Right Corner, Enable it.

Click Load Unpacked on the left and open the Extracted Folder

And you should have revert back to the original twitter instead of “X”
